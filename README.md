# Clone do Pinterest

Este é um projeto de clone do Pinterest desenvolvido com HTML e CSS.

## Visão Geral

Este projeto consiste em replicar a interface do usuário do Pinterest, incluindo os cards de imagens e a disposição em grade.

## Tecnologias Utilizadas

- HTML
- CSS

## Estrutura do Projeto

O projeto possui uma estrutura simples com os seguintes arquivos:

- `index.html`: Contém a estrutura HTML da página.
- `styles.css`: Arquivo de estilo CSS para estilizar a aplicação.

## Como Usar

Baixe ou clone o repositório em seu dispositivo.
Abra o arquivo `index.html` em seu navegador web.

## Personalização

- Modifique o arquivo `index.html` para adicionar ou alterar o conteúdo conforme necessário.
- Adapte o estilo no arquivo `style.css` para atender às suas preferências visuais.

## Pré-requisitos

- Certifique-se de ter um navegador web instalado para visualizar o clone do Pinterest.
